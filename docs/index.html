🔬 Autopsia

ESLint te dice si tu código está bien escrito. Autopsia te dice si tu arquitectura está bien construida. Auditor de Clean Architecture para React Native / TypeScript.

$ npx autopsia-rn init # detecta tus capas
$ npx autopsia-rn scan . # audita el proyecto
$ npx autopsia-rn scan . --html --open # grafo interactivo
🕸 Demo interactivo del grafo el visor de --html, en vivo 📖 Getting started de cero a tu primer scan, paso a paso 📏 Reglas código malo, fix y por qué importa ⚙️ Configuración capas, severidad, baseline ⭐ Repositorio en GitHub código, issues, contribuir 📦 autopsia-rn en npm npx autopsia-rn, sin instalar
